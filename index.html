HTML 注释标签

您能够通过如下语法向 HTML 源代码添加注释：

实例

<!-- 在此处写注释 -->

注释：在开始标签中有一个惊叹号，但是结束标签中没有。

浏览器不会显示注释，但是能够帮助记录您的 HTML 文档。
